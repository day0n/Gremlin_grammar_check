# Gremlin语法检查器

这是一个基于ANTLR4的Gremlin语法检查工具，可以验证Gremlin查询语句的语法正确性。


## 安装依赖

```bash
pip install -r requirements.txt
```

## 使用方法


RUN
```bash
python gremlin_checker.py
# 然后输入查询语句，按Ctrl+D结束输入
```
